1. Make sure you have NodeJS downloaded https://nodejs.org/en/

2. Navigate to “nodeApp” folder in terminal

3. Type “npm install” in terminal to install node modules

4. Type “npm start” to start local server

5. go to your web browser and type “http://localhost:3000/“ in your browser bar.

6. You should be able to see the new page.

--------------------------------------------

If you are experimenting, use npm install _____ to add a new module.
Do not touch bin/www or app.js
